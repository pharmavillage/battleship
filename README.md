## Getting Started

Ready to dive in? Here's how to get started:

1. **Clone the repo:**

   ```sh
   git clone https://github.com/pharmavillage/pharmavillage.git
   ```
2. **Navigate to the project directory:**

   ```sh
   cd pharmavillage
   ```
3. **Install dependencies:**

   ```sh
   npm install
   ```
4. **Run the development server:**

   ```sh
   npm start
   ```
